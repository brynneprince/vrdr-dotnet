◆ CauseOfDeathEntityAxisList

List<(string Line, string Position, string Code)> VRDR.CodingResponseMessage.CauseOfDeathEntityAxisList
get

Entity axis cause of death coding as a flat list. Provided as an alternative to the CauseOfDeathEntityAxis property which groups cause codes by line, CauseOfDeathEntityAxis.

Each entry in the list is a tuple with three values:

  • Line: CauseOfDeathEntityAxisEntry.LineNumber
  • Position: Sequence of code within the line
  • Code>: ICD code
