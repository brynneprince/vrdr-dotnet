VRDR.IJEMortality Class Reference

A "wrapper" class to convert between a FHIR based DeathRecord and a record in IJE Mortality format. Each property of this class corresponds exactly with a field in the IJE Mortality format. The getters convert from the embedded FHIR based DeathRecord to the IJE format for a specific field, and the setters convert from IJE format for a specific field and set that value on the embedded FHIR based DeathRecord. Detailed Description

A "wrapper" class to convert between a FHIR based DeathRecord and a record in IJE Mortality format. Each property of this class corresponds exactly with a field in the IJE Mortality format. The getters convert from the embedded FHIR based DeathRecord to the IJE format for a specific field, and the setters convert from IJE format for a specific field and set that value on the embedded FHIR based DeathRecord.
